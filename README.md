# Wheel of Names

A simple web app that randomly picks a name using an interactive spinning wheel.

## Features

- **Names**: Add up to 100 names. Each name can be removed with the Ã— button.
- **Prize Money** & **Charity Money**: Both start at $0 and increase by **$5** for each name you add.
- **Wheel**: Add at least 2 names to enable the spin button. Click **Spin the wheel** to pick a random winner.

## How to run

Open `index.html` in a browser (double-click or drag into a browser window). No server or build step required.
